SpinWheel

SpinWheel

  • Code language: C#
  • Game engine: Unity
  • Platform: PC
  • Project type: Personal

SpinWheel the game of winners. Game will be played by coins. When you win you will earn more coins or if you are unlucky guy and there's no matches you lose coins. Also you can gain coins in the Daily Reward it will be available in every 12 hours. Bet can be 100, 500, 2000 or 5000 coins. SpinWheel include music and win-effect sound which volume you can controll in settings. I made all in this project programming, graphics, gameobjects etc.
